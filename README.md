# Mac-Space-Period (MSP)


A lightweight AutoHotkey v2 script that brings the macOS/iOS double-tap spacebar shortcut to Windows. Double-tapping the spacebar quickly inserts a period followed by a space to seamlessly start a new sentence.

## Features

* **macOS Style:** Double-tap space to insert a period and a space.
* **Fast-Typing Friendly:** Optimized to handle rapid typing without breaking or misfiring.
* **Smart Repeat Prevention:** Holding down the spacebar will not spam periods.
* **Lightweight:** Uses minimal system resources and runs silently in your system tray.

## Installation & Usage

Choose one of the two methods below to get started:

### Method 1: Run the Compiled Executable (Easiest)
*No installation or AutoHotkey setup required.*
1. Go to the **Releases** section of this GitHub repository.
2. Download the latest `mac-space-period.exe` file.
3. Double-click the downloaded file to run it.

### Method 2: Run the Raw Script (Recommended)
*Perfect if you want to inspect, audit, or customize the code yourself.*
1. Download and install [AutoHotkey v2](https://www.autohotkey.com/).
2. Download the `mac-space-period.ahk` file from this repository.
3. Double-click the `.ahk` file to run it.

### Run on Startup (Optional)

If you want the script to run automatically every time you turn on your PC:
1. Press `Win + R` to open the Run dialog.
2. Type `shell:startup` and press **Enter**.
3. Create a shortcut of your `mac-space-period.ahk` file and drop it into that Startup folder.
